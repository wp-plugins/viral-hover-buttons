Adds social sharing buttons overlay in your pictures and videos when the users hover on them.



== Description ==

A wordpress plugin that displays social sharing buttons in your pictures and videos when the users hover on them. Your articles are guaranteed to get more shares and make you more money as a result.



== Installation ==

1. Install as regular WordPress plugin and activate it.<br />
2. Go to the plugin options page via "Settings" --> "Viral Hover Buttons" (sub)menu.<br />
3. Set your preferred plugin options.<br />

After you activate the plugin, all pictures and videos on your posts and pages will display the social sharing buttons when the user hovers the mouse over it.
